# Gentoo Overlay

## Description

Personal Gentoo overlay

## Problem

Packages not available in the official Gentoo tree or GURU overlay

## Goals

Clean, CI-verified ebuilds that install correctly on ~amd64 systems

## Non-Goals

No stable keywords; no support for architectures other than amd64

## Constraints

EAPI 8; ~amd64 only; masters=gentoo; must pass pkgcheck QA and build tests
